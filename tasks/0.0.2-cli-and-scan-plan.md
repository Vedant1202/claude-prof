## Overview

Strengthen the CLI foundation, then ship the first new command through it. One enabler (a zero-dep
**command registry**) plus two features: CLI-standards polish (per-command help, uniform `--json`,
stdout/stderr split + `--quiet`, completions) and `cprof scan` (standalone secret gate reusing
`leak-check.ts`). Rollback / ledger v2 / drift are out of scope (next spec).

## Architecture decisions (from spec ¬ß4)

- **D1 ‚Äî command registry, zero-dep.** A declarative `Command[]` table drives dispatch, help, and
  completions from one source. Keep each command's `run(flags, opts)` signature and its own
  positional parsing; no CLI framework. The Task 1 refactor is **behavior-preserving** ‚Äî the existing
  CLI suite must stay green unchanged.
- **D3 ‚Äî `--json` envelope.** Every command emits one stdout object `{ command, ok, ‚Ä¶ }`. No
  schema-stability promise in alpha.
- **D4 ‚Äî defer color.** Land stdout/stderr split + `--quiet` only; no color machinery yet (nothing to
  gate today).
- **D5 ‚Äî `scan` takes files**, raw-byte scan, no dir/glob walking.
- **D6 ‚Äî exit codes** 0 clean / 3 leak / 2 not-found / 1 usage; findings carry `line:col`.
- **D7 ‚Äî ship `scan` as-is**, documented "same limits as redaction," `--strict` held in reserve.

## Grounded current state (recon)

- Dispatch is an `if`-chain in `main()` ([cli/src/index.ts:85](../packages/cli/src/index.ts)); command
  list lives only in the `USAGE` string (lines 22‚Äì43). `--help`/`-h` is matched **anywhere** before
  dispatch (lines 73‚Äì81) ‚Äî the per-command-help bug.
- Tests call `main(argv, { cwd, homeDir, stdout, stderr, env })` with `createWritable()`; key
  invariants to preserve: `--help` overview contains `"Usage: cprof <command>"` + `"init"` + `"install"`;
  unknown command ‚Üí exit 1 + `"unknown command: <x>"` + `"--help"` ([cli/test/index.test.ts:36‚Äì61](../packages/cli/test/index.test.ts)).
- `--json` today: `validate`/`diff`/`profiles` only (ad-hoc `flags.includes("--json")`).
- Status logs on stdout to relocate: init `"Wrote claude-profile.json"`
  ([commands/init.ts:73](../packages/cli/src/commands/init.ts)), refresh `"Refreshed‚Ä¶"`, install
  `"Installed N writes"`.
- `leak-check.ts` `OutputLeak` = `{path, tokenIndex, reason}`; token-scan matches expose `match.index`
  (byte offset ‚Üí line:col); the provider-pattern hit (`tokenIndex: -1`) has **no** offset ‚Üí position
  is best-effort for that class.
