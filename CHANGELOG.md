# Changelog

All notable changes to cprof are documented here.

## 0.0.1-alpha.0 (2026-06-21)

### Features

- deep-merge config on install (Wave 2 / F2b) (#2) (bf61eb3)
- implement Claude profile scanning and integrate automatic secret redaction into manifest generation (22370a3)
- add team policy checks (8181e6e)
- track installed profile updates (f579456)
- add registry discovery (d795c1b)
- support remote profile references (01f8192)
- add local profile install (1bfb50e)
- add repomix output file, update gitignore to exclude .tmp, and mark project as WIP in README (dac332e)
- detect global plugin metadata (5db435c)
- add profile diff command (a43eb60)
- add refresh and validate commands (6560039)
- wire init cli variants (625d941)
- add scan report output (3c7f12a)
- add deterministic asset bundler (cd8eac7)
- add manifest builder (f38abe8)
- add source discovery metadata (3475c1e)
- add output leak checks (d025cee)
- add secret redactor (6c78b75)
- add safe traversal policy (6352293)
- add cprofignore policy (d87df13)
- add profile validator api (426f636)
- define phase 1 profile schema (06fa786)

### Documentation

- Docusaurus site + GitHub Pages deploy (T1) (#5) (8b29f51)
- document phase 1 usage (dcd09e1)

### CI

- add phase 1 verification matrix (be4f625)
