# @cprof/schema

The `claude-profile.json` JSON Schema and its TypeScript types, used by
[cprof](https://github.com/Vedant1202/claude-prof) to validate and type the
profiles it produces.

This is an internal building block of the `cprof` CLI:

```bash
npm install -g @cprof/cli@alpha
```
