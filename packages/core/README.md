# @cprof/core

Core engine for [cprof](https://github.com/Vedant1202/claude-prof): scanning,
secret redaction, manifest generation, bundling, schema validation, local
install (deep merge), and profile diffing.

This is an internal building block of the `cprof` CLI. Most users want the CLI:

```bash
npm install -g cprof@alpha
```
